# Codex Integration Notes

This directory contains repository-local agent guidance for Codex-style tools.

## Files
- `SKILL.md` describes when and how an agent should run `prose-hygiene`.

## Why this exists
The project is partly a proof of concept for a two-layer documentation hygiene loop:
- upstream guidance that tells an agent how to behave
- downstream deterministic cleanup and reporting that catches whatever still slips through

If you publish the repo, this folder gives outside readers context for why `SKILL.md` is present and why it matters to the feature-request discussion.
